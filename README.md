# GenPass
🔐 Auto Password Generator Web Application

Auto Password Generator is a modern, secure, and fully customizable web application designed to create strong passwords instantly. In today’s digital world, where online security is essential, this project focuses on helping users generate reliable and complex passwords within seconds. The application provides a simple yet powerful interface that allows users to control password length, choose different strength levels, and optionally include custom words for personalized password creation.

The system is built using core web technologies including HTML, CSS, and JavaScript. The front-end structure ensures a clean and responsive layout, while JavaScript handles the logic behind password generation. The algorithm combines uppercase letters, lowercase letters, numbers, and special characters based on the selected difficulty level, ensuring high randomness and improved protection against brute-force or guessing attacks.

The project emphasizes performance, usability, and clarity. The interface is minimal and distraction-free, making it easy for users of any technical background to generate secure passwords. The responsive design ensures compatibility across desktops, tablets, and mobile devices without losing functionality or visual quality.

Security and randomness are key priorities in this project. Each generated password is dynamically created using programmatic character selection, avoiding predictable patterns. Users can quickly regenerate new passwords until they find one that suits their requirements. Optional customization features allow flexibility without compromising strength.

This project demonstrates practical front-end development skills, DOM manipulation, event handling, and logical structuring of JavaScript functions. It also reflects an understanding of cybersecurity fundamentals, specifically the importance of strong authentication credentials in preventing unauthorized access.

Auto Password Generator serves as both a functional tool and a portfolio project showcasing clean code structure, modern UI practices, and efficient client-side programming. It can be further expanded with features such as copy-to-clipboard functionality, password strength indicators, dark mode support, or backend encryption integration for enhanced security and scalability.
